# Progetto "Crittografia matriciale"

Il progetto è incentrato sulla crezione di un algoritmo di crittografia e decrittografia a matrice che permetta di
inviare in tutta sicurezza i messaggi dal server al client in tutta sicurezza.
L'algoritmo si baserà su una parola chiave scelta dal client per comunicare tuttavia essendo questo un esercizio
puramente a chiave simmetrica la chiave verrà scambiata in chiaro.
